## Installation

```sh
npm install @bemoje/is-window
npm install --save @bemoje/is-window
npm install --save-dev @bemoje/is-window
```

## Usage

```javascript

import isWindow from '@bemoje/is-window'

// fake window object
const window = {}
window.window = window

isWindow(window)
//=> true

```
